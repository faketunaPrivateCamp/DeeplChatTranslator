# Deepl chat translator

## 機能

* チャットの自動翻訳
* 切替可能

## 使い方

### 必要なもの

* 1.19 のPaperもしくはPaperの派生サーバー
* Deepl freeプランのAPIキー


### 手順

1. 一度サーバーを起動します。
2. 停止します。
3. config.ymlがplugins内のプラグイン名と同じフォルダーに生成されるので編集します。
4. apiKey にDeeplのAPIキーを入れます。
5. enabled は有効にするか否かの設定です。 ゲーム内からも変更可能です。
6. チャットを送信すると少しラグがありますが翻訳された内容が送られます。
